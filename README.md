This repository contains a single Java program (`Main.java`) that implements and demonstrates solutions for 10 common Java practice questions. Each problem is implemented as a method and demonstrated in `main`.

Problems included:
1. Reverse a string
2. Check if a string is a palindrome
3. Compute factorial (BigInteger)
4. Compute Nth Fibonacci number (BigInteger)
5. Check primality and count primes up to N (Sieve)
6. Sort an array
7. Find max and min in an array
8. Check if two strings are anagrams
9. Compute GCD and LCM
10. Count vowels and character frequencies

How to compile and run
- Compile:
  ```
  javac Main.java
  ```
- Run:
  ```
  java Main
  ```
